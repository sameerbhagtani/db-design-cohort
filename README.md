# 📦 Database Design Collection

This repository contains a collection of database designs created as part of the **ChaiCode Web Development Cohort, 2026**.

This project is part of my [ChaiCode Web Dev Cohort 2026 Archive](https://github.com/sameerbhagtani/web-dev-cohort-2026). Checkout my entire journey there!

## 📁 Structure

Each database design is organized in its own folder, and typically includes:

- `schema.png` - visual representation of the ER diagram
- `schema.eraser` - ER diagram as code (Eraser format)
- `README.md` - explanation of the design

## 🎯 Purpose

The goal of this repository is to practice and improve database design skills through consistent, real-world problem-solving.
